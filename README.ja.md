# ココイク

「ココイク」は、ユーザーの現在位置から目的地までのGoogleマップリンクを生成し、メールやWebサイトで簡単に共有できるWebアプリケーションです。

## デモ
[デモサイト](https://codeforfukui.github.io/kokoiku/)

## 機能
- ユーザーの現在位置から目的地までの経路を表示するリンクを生成
- URLにパラメータを付加して目的地を指定可能
- 現在位置が取得できない場合は代替の出発地を設定

## 必要環境
- Webブラウザ
- GPS機能のあるデバイス（スマートフォンなど）

## 使い方
1. デモサイトにアクセス
2. 現在位置が自動検出され、目的地までのリンクが表示される
3. リンクをクリックすると、Googleマップで経路が表示される
4. URLにパラメータを付加して目的地を指定することも可能（例: `https://codeforfukui.github.io/kokoiku/#35.950908,136.180781`）
