# kokoiku

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple guide to your current location.

## Demo
[https://codeforfukui.github.io/kokoiku/](https://codeforfukui.github.io/kokoiku/)

## Features
- Detects user's current location and displays a link to directions to a preset destination
- Allows setting a custom destination by appending latitude and longitude to the URL

## Usage
1. Open the [demo page](https://codeforfukui.github.io/kokoiku/)
2. The app will detect your current location and display a link to directions
3. To set a custom destination, append the latitude and longitude to the URL like this: `https://codeforfukui.github.io/kokoiku/#35.950908,136.180781`
